# Basketball Scouted

## Description

Basketball Scouted is a web application designed to simulate what NBA teams might look like if they operated on a global scale, scouting players more akin to European Football than the current NBA draft system. Users, along with 29 other teams from around the world, can send scouts on trips to different cities and sign players in an effort to create the best 5-man lineup.

## Technologies

Basketball Scouted is a dynamic front-end application developed with React, featuring a design crafted entirely in CSS. The application manages state using React-Redux. Additionally, it utilizes the React Google Maps API library to visually showcase the locations of NBA players and teams on an interactive map.
